###easybot###
### Проект простого бота для регистрацие на реальное меропрятие
### Данная версия бота реализована на longpoll и может быть запущена с локального ПК
### Для запуска нужно вставить токен настроенного сообщества vk.com
### Если возникнут проблемы с библиотеками - в файле requirements.txt представлен список библиотек (в файле представлено избыточное количество библиотек, т.к. файл создавался из тестового виртуального окружения)
